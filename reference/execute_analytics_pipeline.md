# Execute analytics pipeline

Execute analytics pipeline

## Usage

``` r
execute_analytics_pipeline(raw_data, config)
```

## Arguments

- raw_data:

  Raw study data

- config:

  StudyConfig object

- study_type:

  Study type ("standard", "endpoints", or custom)

- analytics_package:

  Package containing workflows (optional)

- analytics_workflows:

  Specific workflows to run (optional)

## Value

Analytics pipeline results
