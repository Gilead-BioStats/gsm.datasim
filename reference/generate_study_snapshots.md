# Generate study data snapshots

Generate study data snapshots

## Usage

``` r
generate_study_snapshots(
  study_id,
  participants,
  sites,
  timepoints,
  interval,
  mappings
)
```

## Arguments

- study_id:

  Study identifier

- participants:

  Number of participants

- sites:

  Number of sites

- timepoints:

  Number of timepoints

- interval:

  Time interval

- mappings:

  Technical mapping names

## Value

List of raw data snapshots
